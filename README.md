# Weather-Dashboard-Web-Application
A responsive Weather Dashboard web app, developed using HTML, CSS and Vanilla JavaScript. The application allows users to search for any city and view real time weather information in a clean and interactive interface. 

Features
	•	Search weather by city name
	•	Displays temperature, weather condition, and location details
	•	Unit conversion between Celsius and Fahrenheit
	•	Interactive and user-friendly interface
	•	Dynamic content using DOM manipulation
	•	Stores last searched city using localStorage
	•	Modular JavaScript structure (separate components and utilities)

Technologies Used
	•	HTML5 – Structure of the web application
	•	CSS3 – Styling and responsive layout
	•	JavaScript (Vanilla JS) – Functionality and interactivity
	•	DOM Manipulation – Dynamic UI updates
	•	localStorage – Saving user preferences
	•	Weather API – Fetching real-time weather data

Project Structure
  The application follows a simple modular architecture:
	•	main.js – Entry point of the application
	•	header.js – Header and temperature unit controls
	•	search.js – City search functionality
	•	dashboard.js – Weather display layout
	•	weather.js – API integration and data handling

Purpose of the Project
  This project was developed as part of academic learning to understand how real-world web applications are built using JavaScript. It focuses on concepts such as DOM manipulation, event handling, modular coding, API integration, and responsive UI design.
